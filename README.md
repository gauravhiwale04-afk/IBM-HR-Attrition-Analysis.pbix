 IBM HR Attrition Dashboard
 
Overview
Power BI dashboard to analyze why employees leave (Attrition) using the IBM HR dataset.

Key Insights :
Attrition Rate: 16.12% (Total 237 exits).

Overtime: People doing overtime have 3x higher attrition (30.5%) vs others (10.4%).

Top Role at Risk: Sales Representatives show the highest attrition at 39.76%.

Travel Impact: Frequent travelers are more likely to leave.

Tools Used :
Power BI (DAX, Power Query).

Dataset: IBM HR Analytics (Kaggle).
